# Contributing Guidelines

## كيفية المساهمة | How to Contribute

### الخطوات الأساسية | Basic Steps

1. **Fork** المستودع
2. **Clone** نسختك المحلية
3. **Create** فرع جديد (`git checkout -b feature/amazing-feature`)
4. **Commit** التغييرات (`git commit -m 'Add amazing feature'`)
5. **Push** إلى الفرع (`git push origin feature/amazing-feature`)
6. **Open** Pull Request

### معايير الكود | Code Standards

- استخدم TypeScript
- اتبع Prettier للتنسيق
- أضف اختبارات للميزات الجديدة
- وثّق الكود بشكل واضح

### نوع المساهمات | Types of Contributions

- 🐛 إصلاح الأخطاء (Bug Fixes)
- ✨ ميزات جديدة (New Features)
- 📚 توثيق (Documentation)
- 🧪 اختبارات (Tests)
- 🎨 تحسينات الواجهة (UI Improvements)

---

شكراً لمساهمتك! | Thank you for contributing!
